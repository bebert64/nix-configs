## Docs

`~/.claude/docs/` contains reference material: service descriptions, database URIs, architecture notes, patterns, and gotchas. Consult it when investigating or implementing.

Before starting any non-trivial task, read `~/.claude/docs/README.md` for the index, then read the relevant doc(s).

After changes that affect a service interface, structure, or key patterns, check whether docs need updating and update them.
