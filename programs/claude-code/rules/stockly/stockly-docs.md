## Stockly documentation

Before implementing changes in a Stockly service, consult `~/.claude/docs/`. Read `README.md` for the index, then read the relevant service or pattern doc.

After implementing changes that affect a service interface, structure, or key patterns, check whether docs need updating and update them.

For Rust gotchas (split, deref wrappers, diesel left joins), see `~/.claude/docs/patterns/rust-gotchas.md`.
